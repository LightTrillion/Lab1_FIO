# Lab1_FIO
Лабораторная работа по предмету "Тестирование программных модулей"

Проект: "**Вычисление вида треугольника и координат его вершин**"

Реализован метод, который вычисляет вид треугольника и координаты трех его вершин для отрисовки в поле 100х100 px.
При создании проекта были соблюдены принципы SOLID и были написаны комментарии к коду.
